# PawCare Hub Backend

Initial Spring Boot scaffold for the PawCare Hub backend.

## Structure

- `controller` for REST endpoints
- `service` for business logic
- `entity` for JPA entities
- `repository` for Spring Data repositories
- `config` for backend configuration

## First endpoint

- `GET /api/health`

## Database

MySQL is configured through environment variables:

- `DB_HOST`
- `DB_PORT`
- `DB_NAME`
- `DB_USERNAME`
- `DB_PASSWORD`
- `SERVER_PORT`
